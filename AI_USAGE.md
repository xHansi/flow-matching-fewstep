# AI-Use Statement

As required by the course, this states how AI tools were used in this project.

I used AI coding assistants (chat-based, e.g. ChatGPT / Gemini) as a support tool while
building this project — mainly for boilerplate, looking up PyTorch/API details, debugging, and
tidying up documentation. All the substantive decisions were mine: the research question and
hypothesis, the choice of models (Flow Matching, DDPM baseline, reflow), the architecture and
training setup, the evaluation (which FID metrics, the domain-shift experiment), and the
interpretation of the results. I reviewed and tested the code myself and verified the outputs
before drawing conclusions.

AI was **not** used to fabricate any results — all figures and FID numbers come from actual
training runs on Kaggle, reproducible from the notebooks and scripts in this repository.
